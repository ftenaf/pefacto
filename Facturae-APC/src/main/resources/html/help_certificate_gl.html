Selección de certificado para a sinatura dunha factura "facturae"

Instrucións de uso

  • Seleccionar o certificado co que desexa asinar da listaxe "Certificados dispoñibles".

  • Opcionalmente, pode consultar as propiedades do certificado seleccionado. Aparecen en "Datos do certificado seleccionado". En "Detalles" pode consultar cada atributo concreto.

  • Unha vez seleccionado o certificado, pulse "Continuar" ("Volver" ou pechar ventá para non asinar a factura).


Certificados dispoñibles

  • Emitido para - Propietario do certificado

  • Emisor - Entidade emisora do certificado

  • Data de caducidade - Data máxima de validez do certificado

  • Tipo de certificado - Tipo

Datos do certificado seleccionado

É posible consultar os campos: Versión, Número de serie, Algoritmo de sinatura, Emisor, Data de entrada en vigor (válido dende), Data de caducidade (válido ata), para quen se emite (Suxeito), Clave pública e os diferentes usos.

Detalles

Nesta rexión de texto pódense consultar os detalles dun atributo concreto. Ao seleccionar un certificado e, posteriormente, un atributo deste certificado, o valor detallado do devandito atributo móstrase nesta sección.

Importación / exportación da base de datos

Instrucións de uso

A presente ventá está destinada á importación dunha base de datos externa (con extensión .fedb) ao interior da aplicación e/ou á exportación da base de datos propia (con todas as facturas e borradores almacenados no seu interior) a un ficheiro para a súa posterior importación dende outra aplicación de facturación electrónica.

  • Importar unha base de datos.
    Introduza a ruta onde se localiza a base de datos que se vai importar, así como o nome do ficheiro que a representa. Para iso pode axudarse do botón "Examinar". Lembre que só se poden importar ficheiros con extensión .fedb.
    A continuación, prema en "Importar": aparecerá unha ventá emerxente cunha mensaxe de advertencia sobre a perda de toda a información almacenada ata ese preciso momento na aplicación.
    Se o desexa, pode seleccionar a opción "Gardar borradores ao importar": deste xeito, todos os borradores almacenados na aplicación antes de realizar a importación seguirán estando presentes tras importar a nova base de datos.
    Finalmente, aparecerá unha mensaxe informativa sobre o éxito ou fracaso da importación. Se ten lugar o primeiro caso, terá á súa disposición toda a información (facturas, borradores, documentos adxuntos e arquivos de configuración) contida na nova base de datos. Se, pola contra, o proceso de importación falla, seguirá conservando toda a información almacenada na antiga base de datos.


  • Exportar unha base de datos.
    Introduza a ruta onde desexa localizar o arquivo da base de datos que se vai exportar, así como o nome que quere asignar a este arquivo. Para isto pode usar novamente o botón "Examinar". Lembre que o arquivo terá a extensión .fedb, e que o proceso de exportación non implica ningunha modificación na actual base de datos da aplicación nin a perda de información ningunha.
    A continuación, prema en "Exportar": aparecerá unha mensaxe informativa sobre o éxito ou fracaso da exportación. Se o proceso se leva a cabo satisfactoriamente, atopará un arquivo (.fedb) na ruta seleccionada que posteriormente poderá ser importado dende outra aplicación de facturación electrónica.

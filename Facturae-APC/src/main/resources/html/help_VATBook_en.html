Generating a VAT Book

Instructions for use

  • Select the type of invoices with which you wish to create the VAT Book: invoices issued, invoices received, or both. Set the time interval encompassing the invoice issue date.

  • Click on "Search". All the invoices verifying the previously imposed conditions will be displayed.

  • Subsequently, if the search result is positive, click on "Generate VAT Book". A window will be displayed in which you can select the name and location of the file (in PDF format) that will contain the VAT Book.


Search result.

After setting the search criteria and clicking on "Search", all the invoices meeting these criteria and presenting Value Added Tax (VAT) as a output tax will appear in the table.

All items subject to this type of tax will be displayed for each invoice; the corresponding tax rate will also be shown. The Equivalence Surcharge tax (ES), should it exist, will also be included.

Finally, to sum up the VAT Book, the Total taxable base for the set of all the invoices included will be displayed, as well as the Taxes deriving from applying VAT and the ES (if appropriate) and the Total amount (the sum of the former amounts).

Selección de certificado para la firma de una factura "facturae"

Instrucciones de uso

  • Seleccionar el certificado con el que desea firmar del listado "Certificados disponibles".

  • Opcionalmente, puede consultar las propiedades del certificado seleccionado. Aparecen en "Datos del certificado seleccionado". En "Detalles" puede consultar cada atributo concreto.

  • Una vez seleccionado el certificado, pulse "Continuar" ("Volver" o cerrar ventana para no firmar la factura).


Certificados disponibles

  • Emitido para - Propietario del certificado

  • Emisor - Entidad emisora del certificado

  • Fecha de caducidad - Fecha máxima de validez del certificado

  • Tipo de certificado - Tipo

Datos del certificado seleccionado

Es posible consultar los campos: Versión, Número de serie, Algoritmo de Firma, Emisor, Fecha de entrada en vigor (válido desde), Fecha de caducidad (válido hasta), para quien se emite (Sujeto), Clave Púlica y los diferentes usos.

Detalles

En esta región de texto se pueden consultar los detalles de un atributo concreto. Al seleccionar un certificado, y posteriormente un atributo de este certificado, el valor detallado de dicho atributo se muestra en esta sección.

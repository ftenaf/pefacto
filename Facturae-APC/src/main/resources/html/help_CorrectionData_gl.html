Información de rectificación

Instrucións de uso

Na ventá "Información de rectificación" poderá consultar os datos referentes á rectificación realizada:

  • Factura rectificada: prema en "Ver factura" para visualizar toda a información almacenada sobre a factura obxecto de rectificación.

  • Motivo: pode observar a descrición da modificación efectuada sobre a factura orixinal, así como o código asociado á devandita rectificación.

  • Método: pode observar a descrición sobre o método empregado para realizar a rectificación, así como o código asociado ao devandito método.

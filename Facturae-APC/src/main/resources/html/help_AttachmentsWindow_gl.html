Adxuntos

Instrucións de uso

A presente ventá permite consultar os documentos adxuntos á factura seleccionada, así como a eliminación ou a introdución de novos adxuntos.

  • Para consultar un documento adxunto xa existente, seleccióneo e prema no botón "Abrir ficheiro". A continuación, abrirase o documento e poderá consultar o seu contido.

  • Para borrar un documento adxunto, seleccióneo e prema no botón "Eliminar". Só se poderán eliminar os adxuntos que non estean incluídos no ficheiro XML da factura.

  • Para achegar un novo documento, prema no botón "Engadir". A continuación, abrirase unha nova ventá onde poderá seleccionar o formato do novo adxunto. Deberá introducir, ademais, a ruta onde está o ficheiro que se vai achegar (para iso, se o desexa, poderá usar o botón "Examinar"), así como unha descrición sobre este.


NOTA

Todo documento que se achegue dende a presente ventá non será incluído no ficheiro XML da factura seleccionada. Só se construirá unha referencia interna (dentro da propia aplicación) entre a devandita factura e o documento achegado.

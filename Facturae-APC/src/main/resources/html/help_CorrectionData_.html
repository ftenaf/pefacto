Información de Rectificación

Instrucciones de uso
En la ventana "Información de Rectificación" podrá consultar los datos referentes a la rectificación realizada:

  • Factura rectificada: pulse en "Ver factura" para visualizar toda la información almacenada sobre la factura objeto de rectificación.

  • Motivo: puede observar la descripción acerca de la modificación efectuada sobre la factura original, así como el código asociado a dicha rectificación.

  • Método: puede observar la descripción sobre el método empleado para realizar la rectificación, así como el código asociado a dicho método.
